# Roadmap

## Format & Output
- [ ] Add random numbers (`--number`)
- [ ] Colored output
- [ ] Save to file
- [x] Multiple output formats (kebab, snake, constant, camel, pascal, normal)
- [x] Generate multiple names (`--count`)

## Usability
- [ ] Custom word lists (`--words "file,edit,debug"`)
- [ ] Add adjectives for 3-word combinations
- [x] Prefix/suffix options (`--prefix "my-" --suffix "-app"`)
- [ ] Interactive/repl mode
- [x] Seed mode for reproducible names (`--seed`)
- [ ] Config file for preferences
- [ ] History of generated names
- [ ] Shell completions
- [x] Copy to clipboard (`--copy`)
